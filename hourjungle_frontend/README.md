【前端系統總覽】
整體採用 Dashboard 風格，
側邊欄進行分頁切換，
主畫面顯示對應功能內容。

 1. 主分頁結構Sidebar
-首頁Dashboard
-記帳管理
-報稅管理
-客戶資料管理
-憑證管理
-報表查詢
-系統設定
-使用者管理／權限設定

分頁詳細規劃
1. 首頁總覽（Dashboard）
【卡片資訊】
-本月營收
-當月進項發票金額
-預估稅額
-報稅截止倒數
【圖表呈現】
-每月營收折線圖
-支出類別圓餅圖

------------------------------------------

2. 記帳管理
🔘 子分頁：
-記帳明細列表
-新增記帳
-快速上傳（Excel/CSV）

記帳明細列表：
日期｜類別｜客戶名稱｜摘要｜金額｜發票號碼｜編輯｜刪除

篩選條件Filter
日期區間(Select)
客戶(Select)
類別（進項／銷項）(Select)
關鍵字查詢(Input)

------------------------------------------

3. 報稅管理
🔘 子分頁：
-營業稅報表（401/403）
-二代健保申報
-營所稅計算
-各期稅額試算

功能內容：
-表格試算結果
-自動計算應稅、免稅、零稅率
-上傳報稅 XML / txt
-一鍵生成報表 PDF

 操作按鈕：
[下載報表]
[生成XML檔]
[匯出PDF]
[試算預覽]

------------------------------------------

4. 客戶資料管理
客戶清單表格：
客戶名稱｜統一編號｜地址｜負責人｜聯絡方式｜記帳起始月份｜操作

可搜尋與分類：

關鍵字(Input)
行業類別(Select)
自訂標籤(Select)

新增客戶表單(Button)
資料完整度檢查(Button)
可上傳營業登記證影本等附件(Button)

------------------------------------------

5. 憑證管理
-發票/收據/費用憑證 上傳功能（支援拖曳）
-OCR 辨識（選配）
-憑證分類標籤（例：租金、交通費、雜支）

匯出對帳表功能(Button)

------------------------------------------

6. 報表查詢
可查詢(Search Input)
-營業稅申報表
-應收應付帳款表
-損益表
-現金流量表

支援：

期間查詢(Select)
匯出 Excel / PDF (Button)
*對比前期報表

------------------------------------------

7. 系統設定
-帳務科目設定（自訂）
-稅率設定（5%、0%、免稅）
-憑證分類維護
-Excel 匯入樣板下載

------------------------------------------

8. 使用者／權限設定
-建立事務所內部帳號
-權限群組（主管、會計助理、客戶端登入）
-日誌紀錄查詢(Log)

附加功能建議
-客戶專區登入：提供客戶查詢發票與報表
-通知系統：報稅提醒、報表完成通知
-整合行事曆：繳稅日、申報日標示
